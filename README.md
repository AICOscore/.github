## What is AICO Score?

Like FICO measures creditworthiness (300-850), AICO measures API compatibility with AI agents (300-850).

We analyze APIs across five weighted factors:
- **AI-Readiness** - Can AI agents understand your API?
- **Security** - Safe for AI agent access?
- **Performance** - Fast enough for real-time use?
- **Reliability** - Consistent for 24/7 AI agents?
- **Developer Experience** - Well-documented?

## Why It Matters

By 2027, millions will shop via AI agents like Claude and ChatGPT. Only AICO-compatible APIs will appear in those searches.
